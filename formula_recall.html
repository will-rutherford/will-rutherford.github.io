$\int x^a\,dx=\dfrac{x^{a+1}}{a+1}+C\ (a\ne-1)$
$\int \dfrac1x\,dx=\ln|x|+C$

$\int \sin x\,dx=-\cos x+C$

2 · Integration techniques

By parts: $\int u\,dv=uv-\int v\,du$. Choose $u$ by LIATE (Log, Inverse-trig, Algebraic, Trig, Exp).
u-substitution: set $u=$ inner function whose derivative appears as a factor.
Partial fractions: $\dfrac{1}{(x-a)(x-b)}=\dfrac{A}{x-a}+\dfrac{B}{x-b}$, solve $A,B$ by cover-up.
Improper: swap bad limit for a variable, integrate, take limit. Finite = converges.
FTC + chain: $\dfrac{d}{dx}\int_a^{g(x)}f(t)\,dt=f(g(x))\,g'(x)$.

3 · Series convergence tests

TestRule
nth-term (divergence)$a_n\not\to0\Rightarrow$ diverges. (Cannot prove convergence.)
Geometric $\sum ar^n$converges $\iff|r|<1$, sum $=\dfrac{a}{1-r}$
p-series $\sum\frac1{n^p}$converges $\iff p>1$
Ratio$L=\lim\left|\frac{a_{n+1}}{a_n}\right|$: $<1$ conv, $>1$ div, $=1$ inconclusive
Comparison$0\le a_n\le b_n$; $\sum b_n$ conv $\Rightarrow\sum a_n$ conv
Alternating (Leibniz)$\sum(-1)^n b_n$ conv if $b_n\downarrow0$
Absolute = $\sum|a_n|$ conv. Conditional = $\sum a_n$ conv but $\sum|a_n|$ div. Classic: $\sum\frac{(-1)^n}{n}$.

7 · Subspace / vector-space test

$S$ is a subspace $\iff$ (1) contains $0$, (2) closed under $+$, (3) closed under scalar ×. Most "is it a vector space?" answers are FALSE because $0\notin S$.
